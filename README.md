# 🚀 AI E-Commerce Blog

A modern, responsive, and production-ready blog web application built with **React (Vite)**, **Tailwind CSS**, **ShadCN UI**, and **Framer Motion**.

This project presents a beautifully designed article:

> **"Impact of AI on E-Commerce and Role of Social Media"**

---

## 📌 Features

✨ Clean and modern SaaS-style UI
📱 Fully responsive (mobile, tablet, desktop)
🌙 Dark / Light mode with system detection
🎯 Smooth animations using Framer Motion
📖 Reading progress bar
🧭 Table of contents (sticky sidebar)
🔝 Scroll-to-top button
🔗 Social sharing (Twitter, Facebook, LinkedIn)
💡 Highlighted keywords and quote blocks
⚡ Optimized performance (lazy loading images)

---

## 🧱 Tech Stack

* ⚛️ React (Vite)
* 🎨 Tailwind CSS
* 🧩 ShadCN UI
* 🎬 Framer Motion
* 🎯 Lucide Icons

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/your-username/ai-ecommerce-blog.git
cd ai-ecommerce-blog
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```
